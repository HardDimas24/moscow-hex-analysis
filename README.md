**Разработка сервиса оценки привлекательности строительства в районах Москвы**

Сервис предназначен для аналитической оценки территорий Москвы с точки зрения привлекательности строительства, используя пространственные данные, инфраструктурные показатели и собственные метрики доступности.

Проект использует гексагоны (hex grid) как основную единицу анализа и объединяет данные о парках, школах, метро, инфраструктуре, плотности, и других факторах, формируя комплексный индекс привлекательности застройки.
